Why Connect AI Agents to Knowledge Graphs?

  • LLMs hallucinate – they generate plausible but incorrect answers based solely on training data.
  • Knowledge Graphs provide accurate, structured, up-to-date data (DBpedia, Wikidata, Solid pods, …).
  • Model Context Protocol (MCP) lets AI agents call external tools at runtime — no retraining needed.
  • Goal: expose any Knowledge Graph to an AI agent via a SPARQL-based MCP interface using Comunica.

AI Agent Querying Knowledge Graphs

How It Works

Comunica MCP SPARQL runs as a local MCP server and requires only a single configuration change in tools like Claude, Copilot, or ChatGPT: 

{
  "mcpServers": {
    "comunica": {
      "command": "npx",
      "args": ["-y", "@comunica/mcp-sparql"]
    }
  }
}

Supports Claude Desktop, GitHub Copilot, VS Code, and any MCP-compatible client.

MCP Tools Exposed to AI Agents

query-sparql

Execute SPARQL queries over one or more remote sources: SPARQL endpoints, Linked Data docs, Solid pods, TPF interfaces, HDT files, and more. Supports updates, federated queries, HTTP proxies, and authentication.

query-sparql-rdf

Execute SPARQL queries over a serialized RDF dataset provided as a string. Accepts any RDF media type (Turtle, JSON-LD, N-Triples, …).

Also available: mcp-sparql-solid, mcp-sparql-link-traversal, mcp-sparql-file, mcp-sparql-hdt

Preliminary Findings

  • LLMs effectively query DBpedia and Wikidata — these are well-known during pretraining.
  • LLMs struggle with less-known KGs (e.g. UniProt, DBLP) unless explicitly guided to use them.
  • LLMs can hallucinate SPARQL endpoint URLs and struggle with federated queries.
  • Asking Claude "What movies do Brad Pitt and DiCaprio both play in?" — only 1 answer. With Comunica MCP: 2 accurate answers via DBpedia SPARQL.
  • Future work: guiding agents to KG discovery, exposing schema tools, improving federated queries.

Conclusion

  • Comunica MCP SPARQL enables AI agents to access any public or private Knowledge Graph via SPARQL.
  • A step towards trustworthy, explainable agentic AI grounded in structured knowledge.
